## Glosario de Términos
 
| Término | Significado |
|---|---|
| **RPE** | Rate of Perceived Exertion. Escala del 1–10 de esfuerzo percibido |
| **RPE 8–9** | Dejar 1–2 reps en reserva (no al fallo) |
| **Fallo** | Llevar el set hasta el fallo muscular concéntrico |
| **LLPs** | Lengthened Partial Reps — reps parciales en la posición estirada para extender el set |
| **Myo-reps** | Técnica de rest-pause: set activador al fallo, luego mini-sets de 3–5 reps con 5–10 seg de descanso |
| **Static Stretch (30s)** | Estirar el músculo objetivo de forma estática por 30 segundos como técnica de intensidad |
| **Early Sets** | Todos los sets antes del último. Ligeramente más fáciles (RPE < último set) |
| **Last Set** | El último set de cada ejercicio, llevado al fallo o técnica indicada |
| **ROM** | Range of Motion — rango de movimiento |
